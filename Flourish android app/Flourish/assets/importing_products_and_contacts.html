Example Microsoft Excel and Comma Separated files for Importing Products and Contacts

Product and Contact importing will allow you to import data from Excel files (.xls) and Comma Separated Files (.csv). If you have managed your contacts or products from Excel files or can export the data from another application importing is quick and easy.
 

Supported Fields

The following are the different pieces of data you can include in your import files. No fields are required and you can use any variation of different data in your import file. After uploading the file you will be able to map what each field is what type of data in your file.
 

Contacts

  • First Name
  • Last Name
  • Email Address
  • Phone Number
  • Address Line 1
  • Address Line 2
  • City
  • State
  • Zipcode
  • Country
  • Birthdate
  • Notes
  • Spouse’s Name
  • Spouse’s Birthday
  • Anniversary


Products

  • Product Number
  • Product Name
  • Wholesale Price
  • Retail Price
  • On Hand
  • Target

 

Microsoft Excel (.xls)

Microsoft Excel can be populated with each piece of data in individual cells and each new entry on a new row. Make sure to remove any title rows that don’t contain actual data you want to import. You should only have data you want imported in the file.

Comma Separated Files (.csv)

Comma Separated Files are plain text files with data separated by commas and line breaks. This data will include the following options separated by commas with each individual entry separated by a new line. Make sure to remove any title rows that don’t contain actual data you want to import. You should only have data you want imported in the file.
